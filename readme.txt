== Description ==

[BestMatch](http://bestmat.ch) is a product recommendation application that helps online shoppers find the products that fit them best.
Unlike conventional "filter-based" shopping advisors, BestMatch does not ask users questions such as: PLASMA or LED? Instead, it asks users general lifestyle questions about the in which they will use the product. These questions are easy to answer with even the most basic product knowledge. The usersâ€™ answers, along with the expert knowledge in the BestMatch database, is used to tailor the best possible product recommendation for each customer.
Publishers that install the BestMatch Product Advisor on their website benefit from each purchase made through the advisor by their readers. To learn more about our **monetization** model, go to: http://bestmat.ch/publishers

== Installation ==

1. Upload the bestmatch folder to the /wp-content/plugins/ directory (or use the Wordpress auto-install feature)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. The settings for the plugin can be accessed from the main administration menu.
